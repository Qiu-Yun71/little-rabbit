## 项目简介

基于 Vue3 + Vite + Pinia 构建的一个电商平台项目，实现了从商品浏览、购物车、下单支付及会员中心的全流程功能。

---

## 技术栈

- **前端框架**: Vue 3（Composition API + `<script setup>`）
- **状态管理**: Pinia 3 + pinia-plugin-persistedstate（本地持久化）
- **路由**: Vue Router 5（History 模式）
- **UI 组件库**: Element Plus 2
- **HTTP 客户端**: Axios（二次封装统一请求/响应拦截器）

## 功能模块

### 1. Layout 布局页

- 顶部导航栏（搜索入口、登录入口、购物车数量徽标）
- 吸顶导航（滚动时固定显示）
- 页脚信息区

### 2. Home 首页

- 轮播图（Banner）
- 商品分类入口
- 人气推荐 / 新品首发 / 品牌好货面板
- 热卖商品列表

### 3. 分类页

- **一级分类**：展示大分类及子分类，带 Banner
- **二级分类**：商品筛选（品牌 / 价格区间 / 属性筛选）、排序、分页

### 4. 商品详情页

- 商品图文信息展示
- 规格选择与数量调整
- 加入购物车 / 立即购买
- 相关热卖推荐

### 5. 登录页

- 账号密码登录
- 登录态持久化（Pinia + localStorage）
- 路由守卫控制访问权限

### 6. 购物车页

- 商品列表展示（图片 / 规格 / 数量 / 小计）
- 单选 / 全选 / 数量调整 / 删除
- 合计金额实时计算
- 未登录游客购物车 + 登录后合并

### 7. 结算 / 支付页

- 收货地址管理
- 订单信息摘要
- 提交订单 → 跳转沙箱支付
- 支付结果回调展示

### 8. 会员中心

- 个人信息展示
- 我的订单列表（按状态筛选、分页）
- 订单详情、取消订单

## 核心技术点

- **Axios 二次封装**：统一 baseURL、超时、请求/响应拦截器、错误提示
- **Pinia 模块化**：user / cart / category 三个 store 分离职责，cart 使用持久化插件
- **组合式函数抽象**：`useCountDown`、`useBanner`、`useCategory` 等可复用逻辑
- **路由守卫**：基于登录态控制结算、会员等页面访问
- **全局组件注册**：components/index.js 批量注册全局组件
- **自定义指令**：图片懒加载等扩展指令
